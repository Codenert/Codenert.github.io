Difference between Margin, Border, and Padding

Analogy

When talking about margin, border and padding being applied to an html element is best to think of it as wrapping a gift. In this analogy, the gift or a present referred to as the content or html element, margin, border and padding are 3 diffrent wrapping papers. The first layer represent the padding, second layer represent the border and last layer or outer layer is the margin.

Exploring the differences

There are more than one elements in the html. To list a few here are p, h1, h3, div, span, head, title and etc. You can think of these elements as a box in a web page.
